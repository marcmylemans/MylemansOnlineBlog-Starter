## What Is This Template?

This starter gives you a clean, modern Jekyll blog with:

- **Dark / light theme toggle** — respects your OS preference and saves the user's choice
- **Reading progress bar** — subtle indicator at the top of the page while reading
- **Back-to-top button** — appears after scrolling, smooth-scrolls back
- **Table of contents** — auto-generated from headings, sticky on desktop
- **Related posts** — automatically surfaces posts that share tags
- **Prev / next navigation** — easy browsing between posts
- **Featured post** — the latest post gets a hero slot on the homepage
- **Author bio card** — all driven from `_config.yml`, no code changes needed
- **Schema / structured data** — Article, BreadcrumbList, HowTo (opt-in), Organization schemas
- **Google Analytics** — add your ID to `_config.yml`, done
- **Comments via Giscus** — GitHub-based, privacy-friendly, opt-in

## Writing Your First Real Post

1. Delete this file from `_posts/`
2. Create a new file following the naming convention: `YYYY-MM-DD-your-post-title.md`
3. Copy the front matter from one of the sample posts as a starting point
4. Write your content in Markdown

## Useful Front Matter Options

```yaml
---
layout: post
title: "Your Post Title"
date: 2024-06-15 10:00:00 +0000
categories: [Category1, Category2]   # optional
tags: [tag1, tag2, tag3]             # optional — drives related posts
description: "160-char meta description for SEO"
image: /assets/img/posts/my-image.jpg  # OG image + hero banner
toc: false          # set to false to hide the table of contents
comments: false     # set to false to hide comments on this post
howto: true         # enables HowTo schema (see below)
howto_steps:
  - "Step one: do this"
  - "Step two: do that"
last_modified_at: 2024-07-01
---
```

That's it — go build something great!
